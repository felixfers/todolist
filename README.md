# 📋 Sistema de Avisos por Email (Python + SQLite)

Este sistema envía un recordatorio automático cada hora a **ramiroyuyes@gmail.com** con las tareas pendientes almacenadas en una base de datos local.

## 🛠️ Configuración Inicial
1. **Contraseña de Aplicación:** - Ve a tu cuenta de Google -> Seguridad.
   - Activa "Verificación en 2 pasos".
   - Busca "Contraseñas de aplicaciones" y genera una para "Correo".
2. **Configurar `config.py`:** - Pega tu correo y la contraseña de 16 caracteres generada.

## 🚀 Cómo usarlo
1. Ejecuta el script principal para que empiecen los avisos:
   ```bash
   python main.py
